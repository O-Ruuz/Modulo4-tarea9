Sistema operativo: Es el sofware principal que administra los recursos de 
hardware y permite la ejecución de aplicaciones en un dispositivo. 

Node.js: Es un entorno de ejecución que permite ejecutar códido JavaScript
fuera del navegador.  

Aplicación: Es un programa o software el cual es creado para realizar una
tarea concreta.
 
server.js: Es el archivo principal de configuración y ejecución de un servidor
o aplicación de backend creada con Node.js.
 
Puerto: Es una interfaz virtual en un sistema operativo en el que empiezan y
terminan las conexiones de red. Permiten al dispositivo distinguir que
programa o servicio debe recibir los datos que llegan desde internet. 

Logs: Es un registro secuencial que almacena eventos, actividades y errores
ocurridos en el sistema o servidor.
